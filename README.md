Projeto desenvolvido em JavaScript para cálculo de indicadores de auditoria.

O sistema registra auditorias anuais, calcula o percentual de conformidade de cada auditoria e gera um histórico com média geral e indicação das melhor e pior auditoria.

Funcionalidades:
- registro de auditorias
- cálculo de conformidade
- histórico anual
- média de desempenho
- indicação de melhor e pior auditoria do período avaliado

Tecnologia:
JavaScript
